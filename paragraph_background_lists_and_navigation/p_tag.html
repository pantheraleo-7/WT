Paragraph Tag is a block-level element.

Heading & Pre Tags are also block-level elements and hence are closely associated with the Paragraph Tag.

These 3 Tags are most commonly used for text formatting, often in combination with each other.



A Paragraph Tag always start on a new line, as with all block-level elements.

It always leaves an empty line above and below it.

If texts are written in multiple lines in a Paragraph Tag, they will be displayed in a single line till the entire width of the page gets exhausted. Break Tag should be used to insert line break.

If there are multiple spaces between two texts, it will be reduced to just one.

When there is a need to retain 

       spaces and 
            line breaks, Preformat Tag is used.
It displays text as it is written without any formatting, albeit in monospace style.
